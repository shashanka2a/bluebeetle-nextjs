## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

### Development

```bash
# Run development server
npm run dev

# Run linting
npm run lint

# Type checking
npx tsc --noEmit
```

This project is ready for deployment on:
